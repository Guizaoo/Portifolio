# Portfólio - Guilherme Dias

Portfólio pessoal desenvolvido em React para apresentar minha trajetória, habilidades e projetos atuais como desenvolvedor Full Stack.

## Tecnologias

- React
- React Router
- Tailwind CSS
- JavaScript

## Páginas

- `/`: home com apresentação, habilidades e projetos em destaque.
- `/projetos`: página com projetos atuais, objetivos, stack e previews públicos.

## Rodando localmente

```bash
npm install
npm start
```

## Build

```bash
npm run build
```
